## 🎬 Como instalar

1. Baixe o **instalador** (`.exe`) do jogo na aba **[Releases](../../releases)**.
2. Execute e siga o passo a passo (ele acha a pasta do jogo sozinho).
3. Pronto — abra o jogo em português! 🇧🇷

> 🎥 **Prefere ver um tutorial?** Todo lançamento tem vídeo no nosso **[YouTube](https://youtube.com/@estudioduble)**.
>
> 🛡️ Na 1ª execução o Windows pode mostrar _"O Windows protegeu o seu PC"_ (normal em instalador novo, não é vírus). Clique em **Mais informações → Executar assim mesmo**.
